База података:
  
Табеле:  
  1. Аутомобил
     id - идентификатор (примарни кључ)
     marka - марка аутомобила
     model - модел аутомобила
     cena_iznajmljivanja - цена изнајмљивања датог модела аутомобила по дану
     specifikacije - спецификације датог аутомобила

  2. Локација
     id - идентификатор (примарни кључ)
     adresa - адреса Rent a Car пословнице

  3. Automobil_Lokacija
     id - идентификатор (примарни кључ)
     automobil_id - идентификатор аутомобила (страни кључ)
     lokacija_id - идентификатор локације (страни кључ)
     kolicina - број аутомобила који се тренутно налазе на датој локацији

  4. Корисник
     id - идентификатор (примарни кључ)
     ime - име корисника
     prezime - презиме корисника
     email - имејл адреса корисника
     lozinka - лозинка
     uloga - улога корисника (1 - члан, 2 - запослени, 3 - администратор)

  5. Позајмица
     id - идентификатор (примарни кључ)
     korisnik_id - идентификатор корисника (страни кључ)
     automobil_lokacija_id - идентификатор аутомобила узетог са дате локације (страни кључ)
     datum_preuzimanja - датум преузимања аутомобила
     broj_dana - број дана на колико је изнајмљен аутомобил
     obavljeno - променљива која нам говори да ли је позајмица завршена (има вредност 'DA' или 'NE')

Везе:
  1. Аутомобил <-> Lokacija n:n
  2. Korisnik -> Pozajmica 1:n
  3. Automobil_Lokacija -> Pozajmica 1:n

1. Док је изнајмљивање у току корисник не може изнајмити још један аутомобил (максималан број аутомобила који корисник може да изнајми у једном тренутку је 1)
